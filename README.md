
# 🚀 Запуск BILIMAI в Visual Studio Code

Для того чтобы приложение работало, нужно выполнить эти действия:

### 1. Установите инструменты
*   **Node.js**: Скачайте и установите с [nodejs.org](https://nodejs.org/).
*   **VS Code**: Откройте папку с проектом в редакторе.

### 2. Установка библиотек
Откройте терминал в VS Code (Меню `Terminal` -> `New Terminal`) и введите:
```bash
npm install
```
*Это создаст папку `node_modules`, где будут лежать React и другие инструменты.*

### 3. Настройка API ключа
1. В корне проекта создайте файл с названием `.env`
2. Напишите в нем:
```env
VITE_API_KEY=ваш_ключ_из_google_ai_studio
```

### 4. Запуск сервера
В том же терминале введите:
```bash
npm run dev
```
После этого в терминале появится ссылка, например: `http://localhost:5173`. 
**Зажмите Ctrl и кликните по ссылке** — сайт откроется в браузере и будет работать!

---

### Почему это не работает просто через двойной клик по .html?
Браузеры блокируют выполнение скриптов "с диска" (протокол file://) в целях безопасности. Команда `npm run dev` создает "виртуальный сервер", который обманывает браузер и заставляет его думать, что сайт запущен в интернете, позволяя всем функциям (включая AI и озвучку) работать корректно.
